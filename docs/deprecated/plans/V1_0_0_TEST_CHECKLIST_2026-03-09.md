# V1.0.0 测试清单（全业务模块）

更新时间：2026-03-09  
版本范围：`v1.0.0`（当前已实现功能）

## 1. 使用说明

- 本文档用于手工验收，按“业务模块 -> 功能点 -> 测试项”展开。
- 每条用例执行后请在 `状态` 列填写：`通过` / `失败` / `阻塞`。
- 建议至少准备 3 个账号：`A`、`B`、`C`（其中 `A/B` 可先用白名单 UID）。

## 2. 测试环境

- 前端：`http://localhost:5173`
- 后端 HTTP：`http://localhost:8080`
- WebSocket：`ws://localhost:8080/ws`
- 数据库：PostgreSQL（`5432`）
- 缓存：Redis（`6379`）

建议先确认服务启动：

```bash
docker compose ps
```
